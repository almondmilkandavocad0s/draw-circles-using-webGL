# WebGL Circle Renderer
## Description:
This project demonstrates a basic 2D GPU-accelerated graphics pipeline using WebGL. Users can render and interact with circles on a canvas. The project introduces vertex and fragment shaders, vertex buffer objects (VBOs), and simple physics-based motion with boundary collision detection.
## Key Features:
- GPU-based rendering using WebGL
- Dynamic circle creation and motion
- Color and size attributes per circle
- Real-time updates via JavaScript
- Demonstrates vector math, geometry, and shader-based visualization
## Tech Stack:
- JavaScript + HTML5 Canvas
- WebGL for GPU acceleration
G- LSL shaders for vertex and fragment processing
## Learning Goals:
- Understand the GPU rendering pipeline
- Gain hands-on experience with shaders and VBOs
- Apply linear algebra and vector math in computer graphics
- Explore the fundamentals of interactive WebGL visualizations
